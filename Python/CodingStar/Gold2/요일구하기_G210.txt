년, 월, 일을 입력 받아서
그 날의 요일을 구하여 보세요.

Input
년, 월, 일을 같은 줄에서 입력 받습니다.
단, 1592 < y < 3000, 1 <= m <= 12, 1 <= d <= 31 의 정수입니다.
년도를 제한하는 것은, 계산은 되지만 현재의 달력 체계(Gregorian calendar)에서 바른 결과가 아닙니다.


Output
요일을 구하여 출력합니다.
요일은 'Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat'으로 출력합니다.

Sample Input	Sample Output
2019 4 1 	Mon
