#### 2.함수의 매개변수
- 함수의 매개변수는 매우 중ㅇ요
- 복잡한 내용은 주석으로 함수와 매개변수의 역활을 정리
  - money_sum: 금액의 합, n: 사람 수, arr: 금액 배열
  - def money_sum(m, arr):
- 이는 후에 dp (동적계획법) 배열을 만들때도 마찬가지
- 필요한 것과 편한 것을 넘기기

```python
# money_sum: 돈의 합을 출력
# N : 사람수, money_arr : 사람의 돈
def money_sum(N, money_arr):
    ret = 0
    for i in range(N):
        ret += money_arr[i]
    return ret
```
